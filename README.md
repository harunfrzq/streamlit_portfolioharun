# Bank Customer Churn Dashboard (Streamlit)

Dashboard interaktif untuk menganalisis customer churn pada industri perbankan
menggunakan Streamlit dan Plotly.

## Dataset

Bank Customer Churn Prediction

## Fitur

- Filter negara, usia, churn status
- KPI metrics (Churn Rate, Total Customer)
- Visualisasi interaktif
- Deploy di Streamlit Cloud

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly

## Run Local

```bash
pip install -r requirements.txt
streamlit run app.py
```
